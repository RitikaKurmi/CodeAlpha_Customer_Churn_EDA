# Sentiment Analysis Project

## Objective
To analyze Amazon product reviews and determine customer sentiment.

## Dataset
- amazon.csv

## Tools Used
- Python
- Pandas
- Matplotlib
- TextBlob

## Features
- Sentiment Classification
- Rating Distribution Analysis
- Word Cloud Generation
- Data Visualization

## Output Files
- rating_distribution.png
- sentiment_bar_chart.png
- sentiment_pie_chart.png
- wordcloud.png

## Result
Reviews are classified as Positive, Negative, or Neutral to understand customer feedback.
